# navirpc

mirrors navidrome now-playing onto your discord profile as a "listening to" rich presence: track, artist, album art, live progress bar. headless, no discord desktop client, no selfbot token.

it sets presence through discord's social-layer headless-session api on a real oauth grant. design and the verified mechanism live in the knowledgebase (`~/projects/knowledgebase/navirpc`).

status: not built. the discord side is proven; the navidrome poller and the daemon are next.
